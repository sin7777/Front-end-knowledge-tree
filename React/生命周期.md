# React的生命周期

在之前的版本

* componentWillMount
* componentDidMount
* componentWillUnmount——在卸载组件之前，你不能再设置 state
* componentWillUpdate——不能调用 setState
* componentDidUpdate
* shouldComponentUpdate——不能调用setstate
* componentWillReceiveProps
