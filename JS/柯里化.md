# 函数科里化

在一个函数中首先填充几个参数(然后再返回一个新函数)的技术称为柯里化(Currying）

柯里化是指这样一个函数(假设叫做createCurry)，他接收函数A作为参数，运行后能够返回一个新的函数。并且这个新的函数能够处理函数A的剩余参数。

柯里化(Currying)具有：**延迟计算**、**参数复用**、**动态生成函数**的作用。

Function.prototype.bind = function(){
