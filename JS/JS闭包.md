# 闭包

关键词：执行环境、作用域链、活动对象、立即执行函数、匿名函数

闭包的使用：

* 封装对象的私有属性和方法：闭包通常用来创建内部变量，使得这些变量不能被外部随意修改，同时又可以通过指定的函数接口来操作
* 保护函数内的变量安全：如迭代器、生成器（ES6）
* 在内存中维持变量：如果缓存数据、柯里化（高阶函数的降阶处理）

```JS
var data = [];
for (var i = 0; i < 3; i++) {
  data[i] = (function (i) {
        return function(){
            console.log(i);
        }
  })(i);
}
data[0](); //0
data[1](); //1
data[2](); //2
```
