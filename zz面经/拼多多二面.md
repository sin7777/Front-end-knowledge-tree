## 代码题

长度是98的数组，元素是1-100里的98个，无序，不重复，找出剩下的两个

```JS
function findTwo(arr){
    let arrTmp = new Array(100)
    arr.map( (item,index) => {
        arrTmp[item-1] = 1
    })
    return arrTmp.reduce( (a,b,index) =>{
        if(b === undefined){
            return a.push(index + 1)
        }else{
            return a
        }
    },[])
}
```
